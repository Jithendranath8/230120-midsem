## How to Reproduce

1. Install dependencies:
   ```bash
   cd partB
   pip install -r requirements.txt
   ```

2. Run notebooks in order:
   ```
   task_2_1.ipynb  →  task_2_2.ipynb  →  task_2_3.ipynb  →  task_3_1.ipynb  →  task_3_2.ipynb
   ```
   Tasks 1.x are markdown-only and have no execution dependencies.
   Tasks 3.x depend on data files generated by 2.1 and 2.2.

3. Compile the report (requires LaTeX):
   ```bash
   tectonic report.tex
   ```
